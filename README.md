> **Your complete, beginner-friendly guide to passing the AWS AI Practitioner exam.**
> No prior AI/ML or AWS experience required.

## Exam Overview

| Detail | Info |
|---|---|
| **Exam Code** | AIF-C01 |
| **Level** | Foundational |
| **Duration** | 90 minutes |
| **Questions** | 65 total (50 scored + 15 unscored) |
| **Passing Score** | 700 / 1000 |
| **Cost** | $100 USD |
| **Question Types** | Multiple choice, multiple response, ordering, matching, case studies |
| **Target Candidate** | Up to 6 months exposure to AI/ML on AWS |

## How to Use This Guide

### If you're a complete beginner

1. **Start with Domain 1** ([01-Fundamentals-of-AI-and-ML.md](01-Fundamentals-of-AI-and-ML.md)) — it builds the foundation
2. **Read the "Explain Like I'm 5" boxes** first — they give you the big picture
3. **Try the code samples** — copy-paste them into a Python environment (Google Colab is free)
4. **Look at the ASCII diagrams** — they show how things connect visually
5. **Don't memorize everything at once** — follow the study schedule below

### If you have some experience

1. Skim each domain file for gaps in your knowledge
2. Focus on [07-Key-Comparisons.md](07-Key-Comparisons.md) — the exam loves to test distinctions
3. Use [06-AWS-Services-Cheat-Sheet.md](06-AWS-Services-Cheat-Sheet.md) as a quick reference
4. Review [08-Exam-Tips.md](08-Exam-Tips.md) the day before your exam

## Suggested Study Schedule (3 Weeks)

### Week 1 — Build Foundations (Days 1-7)

| Day | Topic | File | Time |
|---|---|---|---|
| 1 | What is AI/ML, supervised vs unsupervised | [01](01-Fundamentals-of-AI-and-ML.md) (first half) | 1.5 hrs |
| 2 | Neural networks, metrics, ML lifecycle | [01](01-Fundamentals-of-AI-and-ML.md) (second half) | 1.5 hrs |
| 3 | GenAI concepts, tokens, embeddings, LLMs | [02](02-Fundamentals-of-GenAI.md) (first half) | 1.5 hrs |
| 4 | Prompt engineering, RAG, fine-tuning | [02](02-Fundamentals-of-GenAI.md) (second half) | 1.5 hrs |
| 5 | Review Day 1-4, redo code samples | [01](01-Fundamentals-of-AI-and-ML.md) + [02](02-Fundamentals-of-GenAI.md) | 1 hr |

| 7 | Bedrock, SageMaker, Q family | [03](03-Applications-of-FMs.md) (second half) | 1.5 hrs |

|---|---|---|---|
